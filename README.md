
## 🎯 Overview

A deep learning framework for Land Use and Land Cover (LULC) classification using ResNet with Convolutional Block Attention Module (CBAM) and Explainable AI (XAI) techniques. This project leverages the EuroSAT dataset with 5 land cover classes.

### Key Features

- 🧠 **ResNet-CBAM Architecture**: Enhanced feature extraction with channel and spatial attention
- 🔍 **Dual-Perspective XAI**: 
  - *Intrinsic*: CBAM spatial attention maps
  - *Agnostic*: LIME superpixel explanations
- 📊 **Comprehensive Evaluation**: Accuracy, precision, recall, F1-score, confusion matrix
- 🌐 **Interactive Dashboard**: Gradio web interface for real-time inference
- 🚀 **Optimized Training**: Data augmentation, label smoothing, cosine annealing

## 📁 Dataset

**EuroSAT (5 Classes)**:
- AnnualCrop
- Forest
- Highway  
- Residential
- River
